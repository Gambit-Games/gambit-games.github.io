◆ uvCoordinates

Math::Vec2 D6Engine::Resources::AnimationClip::AnimationKeyFrame::uvCoordinates

The UV coordinates of the bottom left corner of the new frame.

A value of (0, 0) represents the bottom-left corner of the sprite sheet , and a value of (1, 1) represents the top-right corner of the sprite sheet.

◆ uvSize

Math::Vec2 D6Engine::Resources::AnimationClip::AnimationKeyFrame::uvSize

The size of the new frame in UV coordinates.

A value of (0, 0) represents a size of 0 (no image will be drawn), and a value of (1, 1) represents the full size of the sprite sheet.

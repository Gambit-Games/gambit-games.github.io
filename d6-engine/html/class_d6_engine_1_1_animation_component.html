bool D6Engine::AnimationComponent::PlayClip ( const std::string &  name,
int  numberOfRows,
int  numberOfFrames,
float  animationSpeed = 5 
)

Plays the animation from SpriteSheet, makes use of PlayClipRange to allow users to just use numberOfRows and numberOfFrames to play the animation NOTE: DO NOT USE THIS TO PLAY ANIMATIONS THAT ARE NOT FULLY FILLED, EG. A 2X4 SPRITESHEET MUST HAVE 8 IMAGES, ANYTHING LESS AND IT WILL PLAY EXTRA BLANKS. TO DO THAT, USE PlayClipRange.

Parameters
namethe name of the spritesheet texture
numberOfRowsnumber of rows in the spritesheet
numberOfFramesnumber of frames per rows in the spritesheet
animationSpeedthe speed of the animation, current default is set to 7.5f, but can be modified by the user

static void UpdateKeyStates ()
 Update all existing Key states. If KEYSTATE is KEY_RELEASE previous frame, update to KEY_FREE for current frame. If KEYSTATE is KEY_DOWN previous frame, update to KEY_HOLD for current frame. This should be called at the START of every frame in the main to update the state of the released keys in the current frame.

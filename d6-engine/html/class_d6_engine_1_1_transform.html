Detailed Description

Position, rotation and scale of an object.

Every GameObject in a Scene has a Transform. It's used to store and manipulate the position, rotation and scale of the object. Every Transform can have a parent, which allows you to apply a position, rotation and scale relative to it.

◆ GetSiblingIndex()

size_t D6Engine::Transform::GetSiblingIndex ( ) const

Gets the sibling index of the GameObject.

If the GameObject (and hence its Transform) shares its direct parent with other GameObjects, they are known as siblings. The sibling index shows the order of this GameObject in the hierarchy of its parent. If the new index equals or exceeds the child count, the GameObject is placed at the end (bottom) of the hierarchy.

Returns
The sibling index of the GameObject.

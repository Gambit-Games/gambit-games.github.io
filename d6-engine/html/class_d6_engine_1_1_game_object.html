◆ SetActive()

void D6Engine::GameObject::SetActive ( bool  value)

Activates/Deactivates the GameObject, depending on the given true or false value.

A GameObject may be inactive because a parent is not active. In that case, calling SetActive will not activate it.

Parameters
valueActivates or deactivates the GameObject, where true activates it and false deactivates it.

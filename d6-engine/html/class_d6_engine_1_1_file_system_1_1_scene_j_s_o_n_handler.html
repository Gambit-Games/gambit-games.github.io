Detailed Description

RapidJSON handler class for scene files.

This class inherits RapidJSON's SAX (Simple API for XML) reader class to walk through the JSON file values and construct a Scene struct based on its contents. The functions here are only meant to be used by RapidJSON (with the exception of BindScene).

For more information on the RapidJSON reader, visit https://rapidjson.org/md_doc_sax.html#Reader.

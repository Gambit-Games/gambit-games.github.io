static

Creates a Prefab and saves it to the filesystem.

The entire GameObject hierarchy is saved during the process.

Example:

auto& pgo = Internal::Core::GameObjects()::Instantiate({ 20, 30 }, 0, { 1.2f, 0.5f }, nullptr, "Test Prefab");
pgo.AddComponent<TestComponent>();
Internal::Core::GameObjects()::Instantiate(&pgo.GetTransform(), "Test Prefab Child Object");
Resources::Prefab::Create(pgo, "../GameAssets/assets/Prefabs/");
Parameters
gameObjectThe GameObject to create the Prefab with.
pathThe path to save the Prefab to.
Returns
The newly created Prefab instance.

GameObject * D6Engine::GameObjectManager::Find ( std::string  name)
static

Finds and returns a reference to a GameObject by name.

For a more performance-friendly alternative that avoids string comparisons, see GameObjectManager.Get.

Warning
GameObject pointers acquired this way are volatile and could be lost during events such as deletion and creation of other GameObjects due to reallocation of the internal storage vector.
Parameters
nameThe name of the GameObject to search for.
Returns
A temporary pointer to the GameObject.

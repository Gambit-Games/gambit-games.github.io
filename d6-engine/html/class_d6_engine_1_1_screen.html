Detailed Description

Provides access to display information as well as a screen-space conversion helper function.

Use the Screen class to get a list of supported screen resolutions or switch the current resolution and window mode.

Math::Vec2 D6Engine::Screen::ScreenPointToNDC ( unsigned  x,
unsigned  y 
)
static

Converts a point on the screen in pixels into Normalised Device Coordinate space.

Pixel space is assumed to start from 0, 0 in the top left corner, while NDC originates from the centre and ranges from -1 to 1, with the bottom left corner being -1, -1.

Parameters
xThe x coordinate in pixel space.
yThe y coordinate in pixel space.
Returns
A vector representing the point in NDC with values from -1 to 1.

Loads a given sound file into the Audio Engine. Call this in your state load function.

Sound unloading is automatically handled on exit.

Parameters
soundPointer to the sound that wants the sound data to be loaded into
clipNameThe identifying name used for this sound clip.
pathPath to the sound file.
modeFMOD loading flags. Determines how the sound is loaded, which is important depending on the size and purpose, as well as if it loops and more. More info here: https://www.fmod.com/resources/documentation-api?version=2.02&page=core-api-common.html#fmod_mode
Returns
True if the sound loaded successfully or already exists, false otherwise.

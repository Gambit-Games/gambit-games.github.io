void D6Engine::Component::SetGameObject ( GameObject *  target)
protectedvirtual

Binds this component to the GameObject after construction.

This function can be overriden to implement additional setup behaviour, however, do remember to call the base implementation to ensure that the component still binds to the GameObject.

As components may not be fully initialised at this stage, getting anything other than the Transform component can produce undefined behaviour.

Parameters
targetPointer to the GameObject hosting this component.

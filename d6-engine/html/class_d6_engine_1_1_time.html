float D6Engine::Time::maximumDeltaTime = 0
static

The maximum value of Time::deltaTime in any given frame.

This is a time in seconds that limits the increase of Time::time between two frames. Defaults to '0' for uncapped deltaTime values.

◆ timeScale

float D6Engine::Time::timeScale
static

The multiplier at which time passes.

This can be used for time slow or speed-up effects. When timeScale is 1.0, time passes at normal speeds. When timeScale is set to 0.5, time passes twice as slow. When timeScale is set to zero, all functions that are frame rate independent will pause. Negative values are ignored.

When manipulating the timeScale, multiply Time::fixedDeltaTime by it to ensure the number of FixedUpdate calls remain constant.

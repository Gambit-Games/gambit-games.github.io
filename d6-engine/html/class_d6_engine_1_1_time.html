◆ timeScale

float D6Engine::Time::timeScale = 1.0f
static

The multiplier at which time passes.

This can be used for time slow or speed-up effects. When timeScale is 1.0, time passes at normal speeds. When timeScale is set to 0.5, time passes twice as slow. When timeScale is set to zero, all functions that are frame rate independent will pause. Negative values are ignored.

Changing the timeScale only takes effect on the following frames. When manipulating the timeScale, multiply Time::fixedDeltaTime by it to ensure the number of FixedUpdate calls remain constant.

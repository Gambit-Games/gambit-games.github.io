Detailed Description

Generates pseudorandom numbers and IDs for gameplay.

The Random class makes use of the random header provided in C++11, using the 64-bit Mersenne Twister Engine to provide a series of uniformly-distributed random number generation features for engine and game usage. The generator is automatically seeded once on startup.

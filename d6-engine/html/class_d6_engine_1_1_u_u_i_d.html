Gets the cached index value of the UUID.

◆ GetCacheVersion()

size_t D6Engine::UUID::GetCacheVersion ( ) const
inlinenoexcept

Gets the cache version of the UUID.

The cache version is used to check if the cache index is still relevant. When a vector is modified, it is possible for the index to no longer point to the correct element. Comparing the UUID's cache version with an internal cache version number will allow users to check if the cached index needs to be refreshed.

Internal cache tracking values should start from 1 so that the cache is initialised on first-time access.

Example:

if (subsystem.uuidCacheVersion > object.uuid.GetCacheVersion()) {
object.uuid.UpdateCache(newIndex, subsystem.uuidCacheVersion);
}
Returns
The cache version of the UUID. '0' means the cache is not initialised.

Creates a vertex array object with the given vertices and indices and binds them.

float vertices[] = {
-1.f, -1.f, 1.f, 0.f, 0.f, 0.f, 0.f, // bottom left
1.f, -1.f, 1.f, 0.f, 0.f, 1.f, 0.f, // bottom right
1.f, 1.f, 0.f, 0.f, 1.f, 1.f, 1.f, // top right
-1.f, 1.f, .5f, .5f, .5f, 0.f, 1.f // top left
// position vertex color uv coords
};
unsigned int indices[] = { 0, 1, 2, 2, 3, 0 };
Graphics::OGLVertexArray* mesh1 = new Graphics::OGLVertexArray(vertices, sizeof vertices, indices, 6);
Parameters
verticesArray of vertices
vSizeSize of the vertices array
indicesArray of indices
iCountNumber of indices in the array
drawTypeThey draw type of the array GL_STATIC_DRAW or GL_DYNAMIC_DRAW or GL_STREAM_DRAW
positionSizeNumber of values for position segment (e.g. 2 numbers for position will equal to 2). Defaults to 2.
colourSizeNumber of values for colours segment (e.g. 3 numbers for colours will equal to 3). Defaults to 3.
texturePositionSizeNumber of values for texture position segment (e.g. 2 numbers for texture position will equal to 2). Defaults to 2.

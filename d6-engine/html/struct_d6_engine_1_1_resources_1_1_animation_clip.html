Detailed Description

Contains SpriteRenderer animation data for Animation components.

The AnimationClip resource is the smallest building block of animation in D6 Engine. They represent an isolated series of frames within a sprite sheet, such as a character's idle, run or attack cycle, and can be manipulated via the Animation component to produce lively results.

Each AnimationClip contains a series of AnimationKeyFrame data structures, which store:

  • The frame number for the key frame to activate
  • The UV coordinates of the bottom left corner of the new frame
  • The size of the new frame in UV coordinates

Sprite sheets follow all the basic requirements of textures, and it is recommended they have either 1 or even numbers of rows and columns.
